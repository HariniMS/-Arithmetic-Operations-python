Part D: Tuning your Models¶

Task 1: Tuning¶

Fine-tuning the alpha parameter of the Bernoulli NB model :¶

We'll be taking various values for alpha and building a model by using each value as the alpha parameter for the model and calculate its accuracy. We'll store all the accuracy values in a list so that they can be used to find that alpha value which gives the best accuracy

In [17]:
alpha = [0.001,0.01,0.1,1,10,100,200,210,230,250,270,300,1000]
accuracy_bern = []

for a in alpha:
    model_bernoulli = BernoulliNB(alpha = a)
    model_bernoulli.fit(X, y)
    y_pred = model_bernoulli.predict(X)
    acc_br = accuracy_score(y, y_pred)
    accuracy_bern.append(acc_br)
    
print(accuracy_bern)

# Hybrid algorithms are used pretty often when we have multiple algorithms and we want to get the 
# cumilative "Best" answer from all the constituent algorithms. This method is called 'Stacking'
# which is an ensemble learning technique. We'll be passing the results we get from the individual
# models(usually refereed to a level-0) to another machine learning algorithm(usually logistic
# regression in case of classification, referred to as level-1) to be trained on

# creating a new dataframe from the results of both the models to feed it to a logistic regression
# model thereby creating a "Hybrid NB" and getting the final results
y_pr = pd.concat([pd.Series(y_pred_br), pd.Series(y_pred_gs)], axis=1)


# Hybrid NB algorithm obtained by stacking both the NB algorithms at level-0 and using the Logistic
# regression algorithm at level-1 to correctly predict which of the models at level-0 are appropriate
# for a particular record
from sklearn.linear_model import LogisticRegression
model_hybrid = LogisticRegression()
model_hybrid.fit(y_pr,y)

# finding the accuracy of the Hybrid model
y_pr_hyb = model_hybrid.predict(y_pr)
acc_hybrid = accuracy_score(y,y_pr_hyb)
print('Accuracy for the Hybrid NB algorithm using default parameters for the base NB models is %0.2f'%acc_hybrid)

Accuracy for the Hybrid NB algorithm using default parameters for the base NB models is 0.82
